Tic Tac Toe

  • Date: September 2015
  • Client: Epicodus project
  • Focus: JavaScript

A web app that allows two players to play Tic Tac Toe against each other — users are allowed to choose the board size from 2×2 to 10×10 grids. It is built using JavaScript, jQuery, HTML, CSS, Bootstrap. MochaJS and Chai Assertion Libraries were used for testing.

Kora

  • Date: October 2015
  • Client: Epicodus project
  • Focus: AngularJS

An app that allows users to post questions for experts to answer. Answered questions can be filtered out from unanswered ones and answers can also be upvoted or downvoted. Order of the answers is arranged according to their number of votes (from high to low). Users are allowed to delete individual answers. Comments can also be added to individual answer. It is built using AngularJS, HTML, CSS, and Bootstrap.

view live example

NIXFLIX

  • Date: October 2015
  • Client: Epicodus project
  • Focus: AngularJS

There are three view pages: the welcome page, the submission page, and the movie reviews page. A "night mode" is added which changes the entire site to a darker theme for reading at night. The list of review titles is sorted numerically and alphabetically. It is built using AngularJS, HTML, CSS, and Bootstrap.

view live example

Face2Face

  • Date: September 2015
  • Client: Epicodus project
  • Focus: PHP

This is a mobile-responsive app that allows users to create user profiles, log in and see other users nearby. They can send each other requests to meet up and in person and the app will generate a close-by location for each pair of users to meet up conveniently. Hashing user passwords is implemented in the database for added security. It is built using PHP, PHPUnit testing, Silex, Twig, HTML, CSS, JavaScript, jQuery, Bootstrap, MySQL and Google Map API.
