Kora

  • Date: October 2015
  • Client: Epicodus project
  • Focus: AngularJS

An app that allows users to post questions for experts to answer. Answered questions can be filtered out from unanswered ones and answers can also be upvoted or downvoted. Order of the answers is arranged according to their number of votes (from high to low) and users are allowed to delete individual answers. Comments can be added to individual answers. It is built using AngularJS, HTML, CSS, and Bootstrap.

Face2Face

  • Date: September 2015
  • Client: Epicodus project
  • Focus: PHP

This is a mobile-responsive app that allows users to create user profiles, log in and see other users nearby. They can send each other requests to meet up and in person and the app will generate a close-by location for each pair of users to meet up conveniently. Hashing user passwords is implemented in the database for added security. It is built using PHP, PHPUnit testing, Silex, Twig, HTML, CSS, JavaScript, jQuery, Bootstrap, MySQL, and Google Map API.
